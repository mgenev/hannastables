Adds Feedburner Subscription Form at the end of post content.

== Description ==

If you want more subscribers then you should place subscription box to the correct place. This plugin adds 
Feedburner Subscription Form at the end of your post content.

1. You can change your Feedburner feed
2. You can apply different CSS to Subscription Form Button according to your Blog theme
3. Very Easy to implement.
4. Just 2 step Process.

== Installation ==

= Step 1 =

1. Download zip file and Extract it 
2. Upload wp-subscriber-form folder to your /wp-content/plugins/ folder.
2. Go to the 'Plugins' page in the menu and activate the plugin.

= Step 2 =

1. Go to Wodpress Admin Panel > Setting > Subscriber Form, set your feedburner feed and other options and update it
